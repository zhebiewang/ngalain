# ngalain


## install angular cli
``` npm install -g @angular/cli ```

https://cli.angular.io/

## install ng-alain
https://github.com/cipchk/ng-alain
https://ng-alain.com/cli/getting-started


```
ng new demo --style less
cd demo
ng add ng-alain
```

## start ng-alain
```
ng serve --hmr
npm run hmr
```
